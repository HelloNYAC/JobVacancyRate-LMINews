The Purpose of This Website

Help user to choose a career/industry

→ Help user to understand which industries have more jobs available by Province

→ User will be able to plan for his/her career based on historical data

→ Help user to explore related Labor Market Information News

→ Targeted user group 1: who needs help to choose a career/industry

→ Targeted user group 2: who is considering a second career

Practice fetching data using Statistics Canada API

→ Statistics Canada has numerous of valuable tables/survies available in their API store

→ Using Job Vacancy Survey Dataset for this project

→ Practice how to fetch data from Statistics Canada API

Practice fetching data using LMI APIs

→ LMI API at GC API Store provides recent News articles related to Labour Market Information

→ News can be filtered by Province, Economic Regions and Time Windows, Keywords etc

→ Familiarize myself with how LMI News
